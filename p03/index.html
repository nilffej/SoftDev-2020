Formerly known as Apache Cordova, PhoneGap is a framework that enables mobile app development using HTML, CSS, and Javascript.

Why use PhoneGap?

  • Uses commonly known languages
  • Creates cross-platform applications: Android, iOS, Windows
  • Offers a native app experience
  • It's capable of implementing native device tech (i.e. camera, contacts, push notifications)

PhoneGap is for you if...

  • you're looking to create simple mobile web apps.
  • you're looking to be time and cost efficient.
  • want an easily deployable product.

Disadvantages

  • iOS development requires iOS Developer membership
  • Unfit for gaming apps
